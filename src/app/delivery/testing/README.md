# Delivery Simulation Tests

This folder documents lightweight simulation tests for delivery flows.

## Scope

- Transporter active flow (`Je prends la route`)
- In-app Leaflet navigation panel state handling
- Farmer tracking flow (`Simuler suivi reel`)
- Live simulation controls and progress behavior on both screens

## Run

```bash
npm run test:sim
```

The command runs only `*.simulation.spec.ts` files in headless mode.
